# in ear detect

Aplicación móvil desarrollada por **Melquisedec Sarfeliz** que detecta notas musicales y tonalidades en tiempo real usando el micrófono de tu dispositivo.

## Descripción

*in ear detect* te ayuda a identificar la nota que estás cantando o tocando y sugiere posibles tonalidades musicales. Es ideal para músicos, estudiantes o cualquier persona que quiera entrenar su oído de forma práctica.

## Características principales

- Detección de notas en tiempo real.
- Indicador cromático de las notas detectadas.
- Sugerencia de tonalidades más probables.
- Historial de notas recientes.
- Interfaz oscura y minimalista.

## Permisos

La aplicación solicita acceso al **micrófono** únicamente para analizar el audio de forma local. No se graba ni se transmite ningún audio ni dato personal.
